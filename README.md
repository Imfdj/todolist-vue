

## todolist-vue

## 技术栈

- 前端： vue + vue-router + vuex + axios + element-ui + less  

- 后端： express + mysql + mocha  [github地址](https://github.com/Imfdj/todolist-express)


## 运行

- 克隆仓库 ``` git clone https://github.com/Imfdj/todolist-vue.git ```
- 进入项目 ``` cd todolist-vue ```
- 安装依赖 ``` npm i ```
- 运行项目 ``` npm start ```
